MÔ TẢ CƠ BẢN VÀ CÁCH CÀI ĐẶT

a) Mô tả chức năng:
Ứng dụng minh họa hệ thống điều hướng chuẩn trong Android gồm:
- Bottom Navigation Bar: Chuyển đổi giữa Hộp thư, Thư đã gửi, Hỗ trợ.
- Navigation Drawer: Menu trượt hiển thị thông tin người dùng và liên kết điều hướng.
- Toolbar: Tích hợp nút menu và tiêu đề động theo từng màn hình.

b) Cách cài đặt và chạy:
- Yêu cầu: Android Studio, JDK 11+.
- Các bước:
   + Bước 1: Clone dự án về máy hoặc giải nén source code.
   + Bước 2: Mở Android Studio -> File -> Open -> Chọn thư mục "DogChat".
   + Bước 3: Đợi Gradle đồng bộ xong (Sync Project).
   + Bước 4: Chạy ứng dụng trên máy ảo (Emulator) API 29 trở lên.
